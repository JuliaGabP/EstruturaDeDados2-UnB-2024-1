# Estrutura de Dados 2 - Implementações em C

## Sobre o Projeto  

Este repositório reúne implementações práticas desenvolvidas durante a disciplina **Estrutura de Dados 2 (UnB Gama)**. O objetivo é consolidar conceitos fundamentais de organização, manipulação e eficiência de estruturas de dados por meio da resolução de exercícios e problemas computacionais.   
Todas as soluções foram implementadas em **linguagem C**, com foco em compreensão algorítmica, manipulação de memória e desempenho computacional.

## Objetivos  

-   Aplicar conceitos teóricos de Estruturas de Dados em implementações
    reais
-   Desenvolver raciocínio algorítmico e análise de complexidade
-   Praticar manipulação de ponteiros e gerenciamento de memória em C
-   Resolver problemas clássicos de organização e busca de dados


##  Conteúdos Abordados

O repositório inclui implementações e exercícios relacionados a:

### Árvores

-   Árvores binárias de busca (BST)
-   Inserção e remoção de elementos
-   Busca de valores
-   Cálculo de altura
-   Elementos mínimo e máximo
-   Percursos (pré-ordem, em ordem e pós-ordem)

### Estruturas Lineares

-   Listas encadeadas
-   Inserção e remoção de nós
-   Busca de elementos
-   Mesclagem de listas

### Tabelas Hash

-   Inserção e recuperação de dados
-   Estratégias de armazenamento e acesso

### Algoritmos

-   Ordenação
-   Manipulação de dados estruturados
-   Problemas clássicos de organização e busca

## Estrutura do Repositório

Os arquivos estão organizados como programas independentes, cada um resolvendo um problema específico ou implementando uma estrutura de dados distinta.   
Cada implementação contém: - Definição das estruturas - Funções de manipulação - Testes ou exemplos de uso

## Contexto Acadêmico

Este repositório foi desenvolvido como material de apoio e prática para a disciplina Estrutura de Dados 2, com finalidade educacional e de consolidação de aprendizado.
